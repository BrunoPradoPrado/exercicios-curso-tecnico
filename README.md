Repositório de Estudos – Curso Técnico em Informática (IFSC)

Este repositório tem como objetivo compilar os principais exercícios desenvolvidos e os conhecimentos adquiridos ao longo do curso técnico em Informática realizado no Instituto Federal de Santa Catarina – IFSC, câmpus Chapecó.

O conteúdo está organizado por disciplinas e áreas de estudo, facilitando a navegação e permitindo acompanhar a evolução das práticas realizadas durante o curso. Cada pasta contém códigos, atividades e projetos relacionados ao tema específico, seja em Programação Estruturada, Programação Orientada a Objetos, Desenvolvimento Web, Banco de Dados ou demais componentes curriculares.

O propósito deste repositório é servir como um registro consolidado de aprendizagem, bem como disponibilizar exemplos de código e materiais de consulta para estudos futuros.
