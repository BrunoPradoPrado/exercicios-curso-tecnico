Esta pasta reúne atividades das disciplinas Algoritmos e Estrutura de Dados I e II, agrupadas em um único diretório devido ao volume reduzido de exercícios em cada matéria.

Alguns arquivos são propositalmente mais enxutos, o que se justifica pela simplicidade das tarefas propostas na época — atividades introdutórias realizadas no início do curso, quando meu conhecimento em programação ainda estava em desenvolvimento.

O objetivo deste diretório é registrar essa evolução, preservando tanto as atividades mais simples quanto as mais completas de forma organizada e acessível.
