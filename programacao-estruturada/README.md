Este diretório reúne atividades desenvolvidas durante o curso técnico, utilizando conceitos de Programação Estruturada, com foco especial em lógica sequencial, tomada de decisão, leitura de sensores e controle de atuadores.

Os exercícios desta pasta foram implementados principalmente utilizando Arduino, combinando código-fonte com esquemas básicos de montagem. O objetivo principal é demonstrar o funcionamento de estruturas fundamentais, como:

-Leitura de entradas analógicas e digitais
-Controle de LEDs e outros componentes
-Uso de funções simples
-Implementação de lógica condicional
-Interação através de botões e sensores

Alguns arquivos originalmente faziam parte de um projeto acompanhado de uma estrutura de diretórios mais ampla (como imagens do circuito ou arquivos auxiliares). Para manter o repositório organizado, foi preservado apenas o código essencial e a estrutura mínima necessária.

O objetivo deste diretório é oferecer um registro direto e acessível das práticas de programação estruturada realizadas com Arduino, permitindo consulta rápida sem a necessidade de configurar ambientes adicionais ou reconstruir o projeto original.
