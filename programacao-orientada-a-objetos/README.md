Este diretório reúne diversos exercícios realizados ao longo do curso técnico, utilizando conceitos fundamentais de Programação Orientada a Objetos (POO).
Os arquivos contemplam diferentes linguagens e práticas, como classes, herança, encapsulamento e criação de pequenos sistemas.

Alguns exercícios originalmente dependiam de ambientes específicos para execução (por exemplo, projetos Java com Servlets, configurações de Tomcat, entre outros).
Para manter o repositório mais organizado e evitar a necessidade de múltiplas extensões ou servidores locais, os arquivos que exigiam estruturas externas foram convertidos para .txt, preservando apenas o código para fins de estudo e consulta.

O objetivo deste diretório é servir como um registro claro da evolução dos estudos e facilitar o acesso ao conteúdo, sem a complexidade de ambientes adicionais.
